kernel12

One of my most used projects, the kernel12 project was originally an adventure into the realms of custom Android kernels. Officially, it supports Sony's 2014 Z3 Compact smartphone, running official CyanogenMod 12 and 12.1 builds based on Android 5.0 and 5.1, respectively. The project later branched out to also support Sony's official Android 5.1 and 6.0 releases, as well as their 'Concept for Android' project running on top of Android 6.0.
The original XDA Forums thread, focusing on cm-12.1, can be found here, while the parts of the project supporting official Sony firmware and Concept firmware, dubbed kernel12s and kernel12c, respectively, are detailed in a separate thread on the XDA Forums.

kerneller

Developed as part of the kernel12 project, kerneller is a homebrew replacement for solutions such as AnyKernel, in that it is capable of building boot images from a precompiled kernel, ramdisk, and device tree binary on the fly while being flashed through an Android recovery. The one thing setting it apart from similar implementations is the ability to let the user choose—at installation time—whether they want SELinux enabled or disabled. For more information, read the source code on Github!
